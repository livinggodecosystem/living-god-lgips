# Living God Improvement Proposals (LGIPs)

Living God Improvement Proposals (LGIPs) document significant technical, governance, economic, and ecosystem decisions for the Living God Ecosystem.

Every major architectural change should be proposed, reviewed, and approved through the LGIP process before implementation.

## Categories

* Blockchain
* Tokenomics
* Governance
* Treasury
* Security
* Venture Capital
* Wallet
* DeFi
* Living God Chain
